# Biblioteca
O site funciona a partir de um sistema de carrinho em PHP, portando precisa do apache e do mysql para rodar, tem um sistema onde pode se adicionar e remover livros do banco, a partir de um loging como adiministrador, registra tambem todos que se cadastram e pegam algum livro da biblioteca, reladando atraves de tabelas todos os alunos que não devolveram os livros, todos os livros retirados e todos os alunos cadastrados.
O codigo do banco vem com alguns dados, apenas para ilustrar, então pode ser apagado ou adaptado, ele roda em mysql;
